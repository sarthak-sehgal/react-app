### Setup your own Firebase database
#### 1. Visit the [firebase website](https://firebase.google.com/).

#### 2. Click on "Get Started"
![Firebase Homescreen](https://github.com/sarthak-sehgal/react-app/blob/firebase-setup/Firebase%20HomeScreen.png)

#### 3. Click on "Add Project"
![Add Project](https://github.com/sarthak-sehgal/react-app/blob/firebase-setup/Firebase%20Get%20Started.png)

#### 4. Fill in the required fields and click on "Create project"
![Create Project](https://github.com/sarthak-sehgal/react-app/blob/firebase-setup/Firebase%20Add%20Project.png)

#### 5. Click on "Continue"
![Continue](https://github.com/sarthak-sehgal/react-app/blob/firebase-setup/Firebase%20continue.png)

#### 6. Click on "Database" on the left side panel. Choose "Realtime Database", if prompted. Select "Start in test mode"
![Start in test mode](https://github.com/sarthak-sehgal/react-app/blob/firebase-setup/Firebase%20Start%20in%20test%20mode.png)

#### 7. Copy the database URL you see
![Database URL](https://github.com/sarthak-sehgal/react-app/blob/firebase-setup/Firebase%20Database%20URL.png)

#### 8. Edit the file src -> axios-orders.js
Change the "baseURL" to the URL you copied.
